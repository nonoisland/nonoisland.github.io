領地系統

📗為了減少玩家財物損失，我們建立了領地系統，可防止破壞、偷竊、怪物傷害。

領地建立 : 使用金鋤選取區域，購買土地

- 首先規劃好你想建立的範圍 ， 使用木鋤點擊所需範圍的對角

- 左鍵左上角 (A)，右鍵右下角 (B)，例如:

A X X X

X X X X

X X X B

選取後使用 /res create <名稱> 創建領地

領地價格: 每方塊 1 休憩幣

主要指令
指令
