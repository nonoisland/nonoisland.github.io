領地系統

📗為了減少玩家財物損失，我們建立了領地系統，可防止破壞、偷竊、怪物傷害。

領地價格: 每方塊 1 休憩幣

1. 領地建立 : 使用木鋤選取區域，購買土地

- 首先規劃好你想建立的範圍 ， 使用木鋤點擊所需範圍的對角

- 左鍵左上角 (A)，右鍵右下角 (B)，例如:

A X X X

X X X X

X X X B

選取後使用 /res create <名稱> 創建領地

2. 主要指令

- /res help (幫助)

- /res create <名稱> (創建領地)

- /res tp <領地> (傳送到領地)

- /res info (查看領地資訊)

- /res set <標誌> [true/false] (設定權限)

- /res select vert (選擇從基岩層到天空)

- /res set (設定領地預設玩家權限 GUI )

- /res pset (設定個別玩家權限 GUI (基岩版使用指令))

- /res padd (新增玩家到領地)

3. 限制

- 最大領地數: 999999

- 最大尺寸: 999999×999999

- 高度範圍: -64 到 320

4. 視覺化

- 16格範圍內顯示領地邊界
