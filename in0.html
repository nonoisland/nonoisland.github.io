休憩島伺服器架構 Wiki

📗為了讓玩家了解更多伺服器，我們將會進行一些介紹。

1. 伺服器架構

為了方便管理，休憩島分為3個伺服器架構。

- 大廳伺服器: 玩家進入的大廳。

- 生存伺服器: 主要生存遊戲伺服器。

- 代理伺服器: Velocity 代理，負責玩家連接和皮膚處理。

2. 版本資訊

- Minecraft: 1.21.7 or later version (網站最後更新時間:9月5日)

- 伺服器核心: Paper/Velocity

- 跨平台: 支援 Java 版和基岩版

3. 世界功能
一、 真實季節

- 季節變化系統 (春、夏、秋、冬)

二、 雙門系統

- 同時開關兩扇相鄰的門

三、 連鎖採集

- 可連鎖砍樹和挖礦

- 指令: /sh (開/關連鎖採集)

四、 坐姿系統

- 可以坐在半磚、樓梯、其他玩家頭上

- 指令: /sit (原地坐下)

五、 生怪磚採集

- 使用絲綢之觸鎬子挖掘生怪磚

六、 彈跳墊

- 特殊方塊提供彈跳功能

七、 鐵砧染色

- 使用鐵砧為物品名稱添加顏色

- 格式: &4&l大劍劍 (紅色粗體)

八、 物品展示

- 在聊天中展示手中物品

九、 生物捕捉

- 捕捉蛋系統，可合成一次性或重複性

十、 農業升級

- 不同工具的耕地效果和其他農業功能

4. BOSS 戰鬥
終界龍獎勵

- 龍血量: 200

- 重生時間: 1個月

- 獎勵: 所有參與玩家均可獲得 2000 coins + 1000 XP
